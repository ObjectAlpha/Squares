Diagonal Squares — 40×40

Правила (кликните, чтобы раскрыть)
  • Поле: 40×40, изначально пустое.
  • P1 ходит первым. Ходы по очереди.
  • Правило диагонали: ставить в клетку нельзя, если у неё есть занятые диагональные соседи (по углам). Примыкание по сторонам разрешено.
  • Очки за ход: 1 + количество занятых ортогональных соседей (вверх/вниз/влево/вправо).
  • Если у текущего игрока нет легальных ходов — он пасует, ход переходит сопернику. Два паса подряд завершают игру.
  • Побеждает игрок с большим счётом; при равенстве — ничья.

Замечание: в браузерах исполняется JavaScript (не Java). Этот проект — чистый HTML/CSS/JavaScript и работает офлайн.
