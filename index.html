Diagonal Squares

P1: 0 P2: 0 ⏱ 00:00 До усиления: — Ход: P1 Режим
Правила (кликните, чтобы свернуть/развернуть)
  • Поле: от 5×5 до 40×40, изначально пустое. P1 ходит первым. Ходы по очереди.
  • Базовое правило диагонали: ставить нельзя, если у клетки есть занятые диагональные соседи (по углам).
  • Очки за ход: 1 + число занятых ортогональных соседей (вверх/вниз/влево/вправо).
  • Пас: если у текущего игрока нет легальных ходов — он пасует. Два паса подряд завершают игру.
  • Первый ход P2: не может быть вплотную по стороне к первому ходу P1 (диагональ и так запрещена).
  • Периодическое усиление: каждые N секунд (N=5..10, задаётся в шапке перед стартом) активируется дополнительный запрет: игрок, чей ход в момент срабатывания, не может поставить фишку в клетку, у которой есть ортогональные занятые соседи. На панели показан обратный отсчёт «До усиления» — когда он достигает 0, чип краснеет, усиление действует до любой постановки фишки, затем сбрасывается и сразу начинается новый отсчёт.
  • Завершение: игра заканчивается после двух пасов подряд, либо по кнопке «Завершить» (в этом случае победитель определяется по текущему счёту).
  • Блокировка настроек: после запуска партии нельзя менять: размер поля, интервал усиления, режим, уровни ИИ — до конца игры.
